## 🔍 What is PyrsistenceSniper?

PyrsistenceSniper is a simple tool designed to help you find persistence threats on your Windows computer. We mixed the original PersistenceSniper with Python to make it easier to use. The name is intentionally misspelled, but the tool works reliably.

Persistence threats are programs or scripts set to start automatically without your knowledge. This tool helps you spot those so you can keep your system safe.
